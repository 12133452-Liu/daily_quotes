
# Daily Quotes (Flutter App)

A very simple beginner-friendly Flutter application that shows a random motivational quote from a local list.  
Just press **Next Quote** to see another one!

## Features
- Offline quotes (no internet needed)
- One clean screen
- Random quote generator
- Beginner-friendly Flutter code
